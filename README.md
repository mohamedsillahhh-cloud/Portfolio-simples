## 🚀 Como Usar

### 1. Clonar o repositório
```bash
git clone https://github.com/mohamedsillahhh-cloud/Portfolio-simples.git
cd Portfolio-simples
```

### 2. Abrir o site
- Abra o ficheiro `templates/index.html` no seu navegador
- Ou use um servidor local (recomendado):

```bash
# Usando Python 3
python -m http.server 8000

# Ou Node.js
npx http-server
```

### 3. Navegar
- **Início** - Apresentação e informações gerais
- **Sobre** - Detalhes sobre Mohamed e Erik
- **Portfólio** - Projetos e habilidades
- **Contactos** - Formulário para enviar mensagens
